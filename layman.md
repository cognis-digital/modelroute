modelroute is a local tool that automatically picks the best AI model for your request and switches to a backup if your first choice is unavailable. You tell it what kind of response you need — fast, cheap, or high-quality — and it figures out whether to use a local model on your own machine (like Ollama or vLLM) or a cloud provider (like OpenAI or Anthropic), in that order if you prefer to avoid cloud costs. It translates your request into the right format for each provider, estimates the cost, and hands back a ready-to-send request — no manual configuration of each provider's API needed. It is aimed at developers and teams building AI applications who want to manage multiple model backends from one place.
